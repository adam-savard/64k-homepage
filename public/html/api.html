About the 28k Homepage Project API


You don't have to navigate to this website in order to get data from the 28k-homepage Project.
You can make use of the API and call
it using tools like Postman, curl, etc.

Below are the routes that you can call and the format they are in.

Method Route URL Example Description
GET /page:pageID /page:qRWU5eh_aYxkVeV8f6H80 This route retrieves the page with the specified ID and returns the text.
This can be piped into a command to save directly to a file.


Method Route URL Example Body Format Description
PUT /update /update {
"creatorKey" : "qRWU5eh_aYxkVeV8f6H80",
" html" : "<div> Hello World!</div>"
}		 Using the information in the body, the HTML document with
the specified key is updated using the data in "html"


Method Route URL Example Body Format Description
DELETE /delete /delete {
"creatorKey" : "qRWU5eh_aYxkVeV8f6H80"
}		 Deletes the creator key specified in the body. To prevent fishing for keys,
no information is returned other than the request being submitted.


Method Route URL Example Body Format Description
POST /newDoc /newDoc {
"data" : "<div> Hello World!</div>"
}		 This will post a new document and return the URL plus the creator key as part of the query string.
